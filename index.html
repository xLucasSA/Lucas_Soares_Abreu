Olá, tudo bem?
Meu nome é Lucas, tenho anos, moro em Montes Claros/MG e sou apaixonado por jogos eletrônicos. Essa paixão sempre me impulsionou em direção à escolha de uma carreira na área de tecnologia.



Apesar de ter iniciado minha carreira como engenheiro, estou em processo de transição de engenharia para tecnologia. Acredito que qualquer experiência é importante para me tornar um bom profissional, e no curso de engenharia foi onde aprimorei o pensamento lógico e analítico, além de aprimorar o trabalho em equipe, resolução de problemas e comunicação, características essenciais.



Contudo, em 2023, foi quando decidi realizar essa mudança. Comecei a buscar conhecimentos para ingressar na área tech e foi quando tive meu primeiro contato com a programação, especificamente com o Python. Comecei a utilizá-lo, pois é uma linguagem relativamente fácil de se aprender e muito versátil. Construí alguns códigos para auxilar no meu dia a dia de trabalho, principalmente para análise e tratamento de dados. Logo após, fui contemplado com bolsa de estudos do iFood! O curso foi ministrado pela Cubos Academy, onde desenvolvi projetos em Javascript, Node.js, PostgresSQL, SQL, Git e GitHub, construção de APIs no padrão API REST, entre outras funcionalidades voltadas para o Backend de aplicações web.



Fora isso, venho me dedicado ao aprendizado do framework Django por conta de uma nova demanda do meu trabalho atual, onde estamos com projetos para serem desenvolvidos para nossos clientes como forma de aplicação web e isso tem sido desafiador mas estou gostando muito da experiência e de aprender coisas novas. Também estudo a biblioteca do React e o framework Next.js para o desenvolvimento de projetos pessoais.



Acredito que nessa área o aprendizado é algo constante, por isso já tenho meus próximos conteúdos fixados que irei estudar que são: testes unitários, Docker, AWS, Sawagger (documentação de API’s), Kafka
